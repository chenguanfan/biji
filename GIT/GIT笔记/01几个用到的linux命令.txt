#### vim编辑器

1.打开文件
vim 文件名 #打开文件


2.编辑文本
    i   #进入到内容编辑模式  新手记得这个就行


3.编辑完成后操作
    #3.1 编辑完成后按键盘[ESC],退出编辑模式.
    #3.2.然后输入如下指令,进行相关操作
        :wq  #保存并退出当前文件
        :w   #保存当前文件并留在编辑器中
        :w！ #若文本为只读模式，强制保存
        :q   #退出当前文件（仅在没有未保存更改的情况下）
        :q!  #退出当前文件，忽略任何未保存的修改
